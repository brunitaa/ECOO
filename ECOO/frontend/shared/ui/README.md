# ECOO UI Design System

Este paquete contiene los componentes, tokens y utilidades compartidas del ecosistema frontend.

## Objetivo

- Compartir un lenguaje visual único
- Centralizar componentes de interfaz premium
- Mantener consistencia entre landing, admin, company y citizen

## Contenido

- `src/components` — botones, tarjetas, iconos, navbars, footers y layouts reutilizables
- `src/styles` — tokens de color, tipografía, sombras, gradientes y glassmorphism
- `src/motion` — transiciones y animaciones globales

## Componentes compartidos clave

- `BottomNav` — navegación inferior consistente
- `Navbar` — barra superior con saldo y usuario
- `EcoCoinBadge` — etiqueta de saldo EcoCoins
- `RewardCard` — tarjeta de catálogo de recompensas
- `AppShell` — contenedor base de página

## Uso

Cada frontend puede importar directamente desde `@ecoo/ui`.
