# ECOO Shared Design System

Esta carpeta es el espacio central de la plataforma para la arquitectura visual y de experiencia de usuario.

## Propósito

- Documentar los tokens de diseño globales
- Mantener las reglas de branding corporativo
- Guardar estilos, animaciones, iconografía y patrones de layout

## Estructura sugerida

- `colors/`
- `typography/`
- `spacing/`
- `shadows/`
- `gradients/`
- `animations/`
- `components/`

> El paquete `frontend/shared/ui` contiene la implementación actual del design system.
